DEVELOPING MY PORTFOLIO

ABOUT THE PROCESS

This website has went through a countless amount of iterations in the past year and a half and as a result has become a bit of a passion project. After transitioning from Webflow, I decided I wanted full autonomy of the design so I decided to develop it myself!

Once Figma released developer mode, I wanted to play with the Visual Studio Code plugin to get an intuition for how the design to development process could be streamlined better.

The Figma Visual Studio Code plugin (right) made sectioning out the divs a lot easier whereas before, I would take the time to draw out how I wanted to encapsulate each flex element. The plugin's capability to visualize this made for a much smoother process in coding flex elements. The plugin also made dropping in new hex codes super quick!
